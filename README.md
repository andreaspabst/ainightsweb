# ainightsweb

Neue Version der AI-Nights-Website (**ainights.ai**), gebaut mit [Astro](https://astro.build). Löst die bisherige WordPress/Divi-Seite ab: gleiche Inhalte, Bilder und URLs, modernes dunkles Design.

## Entwicklung

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # statischer Export nach dist/
npm run preview  # dist/ lokal servieren
```

Node 22 (siehe `.nvmrc`).
